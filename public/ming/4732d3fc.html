1：创建docker file

2：创建Jenkinsfile
Jenkinsfile 是jenkins出的用一套用来定义流水线模式的任务的脚本
其中有两种模式

  • pipeline jenkins标准的 配置脚本模式
  • script 使用groovy dsl 实现的jenkins 流水线模式脚本

我采用的是script模式 因为灵活 好用 两者区别请参考jenkins官网

将其中的用{} 包含的中文 替换成相应的参数即可
